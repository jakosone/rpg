This is the instructions for the game RPG 2.

*********Starting the game******************
- Compile and start game:
  1. compile rpg2.asd
  2. evaluate (asdf:load-system :rpg2)
  3. evaluate (rpg)

*********Moving in the game*****************
Type:
W - up
A - left
S - down
D - right
I - swing your sword

********Avatars in the game*****************
@ - Player (you)
H - Potion
X - Poison

The other ones are monsters.

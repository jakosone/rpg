This is the instructions for the game RPG 2.

*********Starting the game******************
- Compile game by compiling rpg2.asd &
  (asdf:load-system :rpg2)
- Run (rpg)

*********Moving in the game*****************
Type:
W - up
A - left
S - down
D - right

********Avatars in the game*****************
Player (you) - P
Potions - H
The other ones are moving monsters.

This is the instructions for the game RPG 2.

*********Starting the game******************
- Compile game by compiling rpg2.asd &
  (asdf:load-system :rpg2)
- Run (rpg)

*********Moving in the game*****************
Type:
W - up
A - left
S - down
D - right
I - swing your sword

********Avatars in the game*****************
@ - Player (you)
H - Potion
X - Poison

The other ones are monsters.

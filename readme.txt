This is the instructions for the game RPG 2.

*********Starting the game******************
- Compile with rpg2.asd
- Run (rpg)
You can re-initialize the game with (initialize-game)

*********Moving in the game*****************
Type:
W - up
A - left
S - down
D - right

********Avatars in the game*****************
Player (you) - P
Potions - H
The other ones are moving monsters.
